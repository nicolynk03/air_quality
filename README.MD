# AQI Air Quality Project
The project's dataset is air quality index of most countries. It is obtained from [Kaggle](dataset\README.MD).

Goals of the project:
* Forecast next-day AQI (regression) (numerical AQU value for tomorrow)
* Explain model decisions to explain what affects AQI most


## Getting Started
In the terminal: enter **pip install -r requirements.txt** to download project's dependencies
